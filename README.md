# Dashboard

## Dependencies
- [Node](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/en/docs/install)

## Installing
`yarn`

## Running
`yarn start`

## Developing components
`yarn storybook`

## Integration developing
`yarn dev`

## Integration developing without typechecking
`yarn dev -t`

## Testing
`yarn test`
